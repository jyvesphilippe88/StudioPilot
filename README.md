# StudioPilot

StudioPilot est une suite Ross Dashboard professionnelle destinée au pilotage d'un Blackmagic ATEM Television Studio HD8 ISO.

## Objectifs
- Interface tactile Full HD
- Director, Scenes, MacroLab et Setup
- Architecture modulaire
- Gestion des mémoires et des macros
- Compatible Ross Dashboard 9.15

## Roadmap
La première étape est le développement de StudioPilot Director avec un noyau de connexion ATEM et une interface tactile optimisée.
